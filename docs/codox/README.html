benchtools

Shared utilities used by other projects that benchmark Clojure and Java code.

Currently very unstable, not appropriate for use as a library. The code itself may have examples that are useful starting points.

Building

Builds with Apache Maven, specifically the clojure-maven-plugin, eg:

mvn clean install
